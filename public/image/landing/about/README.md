# 关于我们图片目录

## 需要的图片文件

请将以下图片文件放置在此目录中：

### 1. about_team.jpg
- **尺寸**: 800×600px (4:3比例)
- **用途**: 团队照片展示
- **内容建议**: AFORE团队成员合影，专业形象

### 2. about_facility.jpg
- **尺寸**: 800×600px (4:3比例)
- **用途**: 设施照片展示
- **内容建议**: 公司办公环境或生产设施

### 3. about_rd_center.jpg
- **尺寸**: 800×600px (4:3比例)
- **用途**: 研发中心展示
- **内容建议**: 研发实验室或技术中心

### 4. about_production.jpg
- **尺寸**: 800×600px (4:3比例)
- **用途**: 生产线展示
- **内容建议**: 产品生产线或质量控制

## 图片要求
- 格式: JPG 或 WebP
- 文件大小: ≤ 400KB
- 质量: 高清，专业
- 风格: 现代企业形象
- 内容: 体现公司实力和专业性
